# CodeAlpha Python Programming Internship

This repository contains three Python projects completed as part of the CodeAlpha Python Programming Internship.

## Task 1 – Hangman Game

A simple text-based game where the player guesses a hidden word one letter at a time. The game ends when the player guesses the word or runs out of six chances.

**Concepts Used:**

* `random` module
* Loops
* If-Else statements
* Lists
* Strings

---

## Task 2 – Stock Portfolio Tracker

A basic program that calculates the total investment value based on stock names and quantities entered by the user. Stock prices are stored in a dictionary, and the result can be saved to a text file.

**Concepts Used:**

* Dictionary
* User Input
* Arithmetic Operations
* File Handling

---

## Task 3 – JPG File Mover

A simple automation script that moves all `.jpg` files from one folder to another. If the destination folder does not exist, it is created automatically.

**Concepts Used:**

* `os` module
* `shutil` module
* Loops
* If-Else statements
* File Handling

---

## Language Used

* Python 3

These projects were developed to practice basic Python programming concepts and improve problem-solving skills through simple real-world applications.
